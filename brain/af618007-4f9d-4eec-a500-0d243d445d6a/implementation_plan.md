# بناء نظام "نسيت كلمة المرور" (Forgot Password Flow)

حالياً، رابط "نسيت كلمة المرور" في صفحة الدخول لا يؤدي إلى أي مكان. لحل هذه المشكلة، سنقوم بإنشاء دورة كاملة لاستعادة كلمة المرور باستخدام Supabase.

## Proposed Changes

### Component 1: صفحات استعادة كلمة المرور
سنقوم بإنشاء صفحتين جديدتين بتصميم متناسق مع صفحة تسجيل الدخول:

#### [NEW] [forgot-password/page.tsx](file:///f:/TKHSAS/app/forgot-password/page.tsx)
- صفحة تطلب من المستخدم إدخال بريده الإلكتروني.
- تقوم بإرسال رابط استعادة كلمة المرور إلى الإيميل باستخدام `supabase.auth.resetPasswordForEmail`.
- تعرض رسالة نجاح تطلب من المستخدم فحص بريده.

#### [NEW] [reset-password/page.tsx](file:///f:/TKHSAS/app/reset-password/page.tsx)
- الصفحة التي يتم توجيه المستخدم إليها عند الضغط على الرابط في الإيميل.
- تحتوي على خانتين: "كلمة المرور الجديدة" و "تأكيد كلمة المرور الجديدة".
- تقوم بتحديث كلمة المرور في قاعدة البيانات باستخدام `supabase.auth.updateUser`.

### Component 2: صفحة تسجيل الدخول
#### [MODIFY] [login/page.tsx](file:///f:/TKHSAS/app/login/page.tsx)
- سنقوم بتحديث الرابط `<a href="#">نسيت كلمة المرور؟</a>` ليصبح رابط فعال يوجه إلى صفحة `/forgot-password`.

## User Review Required
> [!NOTE]
> هذا النظام سيعتمد على خدمة إرسال الإيميلات الافتراضية في Supabase (والتي تسمح بـ 3 رسائل في الساعة كحد أقصى للمشروع المجاني). في المستقبل عند إطلاق المنصة فعلياً، ستحتاج إلى ربط Supabase بخدمة إيميلات مخصصة (مثل Resend أو SendGrid) لرفع هذا الحد وتخصيص شكل رسالة الإيميل.

## Verification Plan

- إدخال بريد إلكتروني مسجل.
- تلقي الإيميل والضغط على الرابط بداخله.
- تعيين كلمة مرور جديدة بنجاح وتجربة تسجيل الدخول بها.
